# Curso Python

Bienvenido al curso de Python. Este repositorio contiene guías y recursos para aprender Python desde cero.

## 📚 Contenido

Este curso incluye las siguientes guías:

### 1. [Instalación de Python y pip](./instalacion%20de%20python%20y%20pip.md)

Guía completa sobre cómo instalar Python y pip en Ubuntu. Incluye:
- Verificación e instalación de Python 3
- Configuración de aliases para comandos
- Instalación y uso básico de pip
- Buenas prácticas con entornos virtuales

### 2. [Introducción a uv (Astral)](./introduccion%20a%20uv%20de%20Astral.md)

Aprende a usar uv, el gestor moderno de paquetes y entornos para Python. Incluye:
- Qué es uv y por qué es más rápido que pip
- Instalación de uv
- Gestión de dependencias con uv add y uv sync
- Diferencias entre pyproject.toml y uv.lock
- Ejercicio práctico completo

## 🚀 Primeros pasos

Si eres nuevo en Python, te recomendamos seguir las guías en este orden:

1. Comienza con la [guía de instalación de Python y pip](./instalacion%20de%20python%20y%20pip.md)
2. Una vez que tengas Python instalado, explora [uv para gestión moderna de proyectos](./introduccion%20a%20uv%20de%20Astral.md)
